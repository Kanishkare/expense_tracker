# Expense Tracker and Budget Analyzer

A complete full-stack application for tracking personal expenses and managing budgets. Built with Spring Boot 3, Spring Security (JWT), JPA, H2 Database, and a vanilla JavaScript frontend.

## Features
- **Authentication**: Secure Signup and Login with JWT and BCrypt password encryption.
- **Expense Management**: CRUD operations for expenses with categories.
- **Budgeting**: Set monthly spending limits.
- **Visual Analytics**: Real-time pie chart showing expense distribution by category.
- **Real-time Notifications**: "Budget Exceeded" warning when spending surpasses limits.
- **User Specific**: Data is isolated per user.

## Prerequisites
- **Java 17** or higher
- **Maven 3.6+**
